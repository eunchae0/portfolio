Project Description

롬앤 제로 벨벳 틴트
제품은 #베이크드시리즈로 가을 무드에 어울리는 립스틱 컬러를 선정하고 '베이킹 하다.' 와 접목하였다. 웹레이아웃은 가을을 책임져 줄 '구운 컬러의 립스틱을 바르고 피크닉을 가다.' 라는 컨셉으로 컬러 조합을 구성하였으며, 감성 피크닉에 어울리는 레드&화이트 컬러의 체크 패턴 피크닉 매트를 메인에 깔아주고 폴라로이드 사진을 디자인 요소로 활용하였다.

Project Date 2021.07
Role 웹 레이아웃
Client 여름이 끝나고 가을 무드의 립스틱을 찾는 고객
